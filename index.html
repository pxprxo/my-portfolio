During my Master's in IT, I was involved in various projects related to my coursework. I applied classroom knowledge to develop projects into real-world. It was an invaluable experience that allowed me to immerse myself in a simulated tech industry.

Here are some of the major projects...

Spilt Bill Application Design

This product design project is part of my personal project, created to showcase my UI/UX design skills using Figma.
I designed this application inspired by a worksheet
I previously created to manage group expenses more clearly. The idea came from real-life situations where splitting bills with friends whether during meals, shopping, or travel—often turns into a confusing task.

Committee/Board Management System

I took on the role of Business Analyst in this Master's project. My responsibilities included gathering client requirements and translating them for the development team to develop the system through DevOps principles on Azure. Prepared SRS documents, wrote User Stories, diagrams, and designed UX/UI wireframes.

Also communicated with the team and presented the project to clients. Recived positive feedback with
"Good user friendly!"

SEA Tourist Web Application

This project was developed during Cloud Computing course. I collaborated with a team to create a web application using Node-RED and various APIs. I learned the principles of this technology and applied them to develop a Currency Converter.
Used Nodes to connect them and used JavaScript to define the functions for the Nodes working efficiently.

Cinema Booking UI Testing

This project applied knowledge from the Software Testing and Quality Assurance course to design and define a detailed Software Testing strategy.
I utilized Black-box Testing principles to verify the accuracy and compliance with Software Requirements specifications as well as prepared comprehensive test result documentation.
